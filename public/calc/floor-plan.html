📋 როგორ გამოვიყენო

1. დაწერე prompt ან აირჩიე template.
2. დააჭიე გენერაცია — AI დააბრუნებს FloorspaceJS JSON-ს.
3. FloorspaceJS editor-ში: File → Import Floorplan → აირჩიე ჩამოტვირთული plan.json.
4. შეცვალე, დაამატე spaces-ი, stories-ი → 3D preview მოდით-ზე.

🔗 Export-ი calc-ებში

FloorspaceJS აგროვებს space area + thermal zone + story height. ეს metrics შეიძლება გადავიდეს:
→ heat-loss (EN 12831) — volume, wall area
→ hvac — occupied area, Qvent
→ floor-pressurization — corridor geometry
